# **RAY.NVIM**
A simple, light-weight, and a good starting point for neovim customization. This
config provides some popular NeoVim plugins pre-configured while keeping the
customizbility of neovim intact.
This started as my own personal config, is now made public for the people who
want to get started with neovim.

## Installation:

```bash
git clone https://github.com/RayAntariksha/Ray.nvim.git ~/.config/nvim && nvim
```
